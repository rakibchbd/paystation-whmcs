# WHMCS PayStation Payment Gateway Module  
![License](https://img.shields.io/badge/License-MIT-green.svg)
![WHMCS](https://img.shields.io/badge/WHMCS-8.x-blue)
![PayStation](https://img.shields.io/badge/PayStation-BD-red)

A custom WHMCS payment gateway module that integrates **PayStation (Bangladesh)** with your WHMCS billing system.  
This module allows your customers to pay invoices securely through the PayStation checkout system.
